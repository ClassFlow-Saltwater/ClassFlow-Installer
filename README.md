# ClassFlow Desktop 🖥️

ClassFlow Desktop is a native macOS application engineered for ultimate speed and fluid user experiences. By leveraging the power of **Tauri** and **Rust**, it wraps the ClassFlow web platform into a lightweight desktop environment with blazing-fast rendering and ultra-smooth animations.

## ✨ Features
* **Native Desktop Experience:** Runs independently as a native macOS window.
* **Fluid Animations:** High-performance rendering for custom, smooth UI transitions.
* **Lightweight Architecture:** Powered by Tauri, ensuring a tiny app footprint compared to traditional desktop wrappers.

## 🛠️ Built With
* **Frontend:** HTML5 / CSS3 / JavaScript (with Vercel deployment)
* **Desktop Framework:** [Tauri v2](https://tauri.app/)
* **Backend Runtime:** [Rust](https://www.rust-lang.org/)
